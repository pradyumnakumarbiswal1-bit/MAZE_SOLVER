# MazeSolver

MazeSolver is a Python-based pathfinding application that finds the shortest path through a maze using the A* (A-Star) search algorithm. It visually demonstrates how the algorithm explores nodes, calculates the optimal route, and efficiently navigates from the start point to the destination. Built with the Pyamaze library, this project is an excellent learning resource for understanding AI search algorithms, graph traversal, and pathfinding techniques.

## Features
- Finds the shortest path using the A* algorithm
- Visualizes node exploration and path traversal
- Interactive maze generation with Pyamaze
- Efficient heuristic-based pathfinding
- Beginner-friendly implementation for learning AI concepts

## Technologies Used
- Python
- Pyamaze
- A* (A-Star) Search Algorithm
- Priority Queue (Heapq)

## Output
The application visually displays:
- Maze generation
- Search process
- Optimal path from source to destination
